WHAT IS CSS?

CSS stands for cascading style sheets, which is the language used to determine the appearance and format of website pages. With CSS, you can set the font type, text color, and page background. CSS is used together with markup languages, such as HTML and XML to build an attractive website that functions well. With CSS, you simply write code once for an HTML element to apply to all pages. Later, when you want to make changes, you also only need to make changes to that one code. Practical, right?
